# Naive Bayes Intuition Visualization

I built this project to understand the basic idea behind Naive Bayes in a simple visual way.

In this project, I used a small spam and ham example and showed how Naive Bayes starts by looking at class probability and word patterns.

I created three parts in the visualization.

The first part shows the prior probability of each class using a pie chart. In this example, I used spam as 40% and ham as 60%. This helps show that Naive Bayes first looks at how common each class is.

The second part shows a word cloud for spam words. These words represent the kind of words that are more likely to appear in spam messages.

The third part shows a word cloud for ham words. These words represent the kind of words that are more likely to appear in normal or non-spam messages.

This project does not build a full Naive Bayes classifier. It is mainly for understanding the intuition behind how Naive Bayes separates classes based on probability and word patterns.
